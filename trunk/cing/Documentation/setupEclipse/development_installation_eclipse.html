Install the requirements:

  1. Install Fink
    Make sure you have the unstable trees in your fink.conf too. I had to add: 
    /unstable/main and /unstable/fink
    and do: 
    fink selfupdate
  2. Install Matlibplot plotting utilities 
    fink install matplotlib-py25
    Note that this also installs the python 2.5 numeric dependencies.
  3. Install cython 
    fink install cython-py25
  4. Install gawk 
    fink install gawk
  5. Install ImageMagick (for convert program) 
    fink install imagemagick
  6. (Optionally) install IPython 
    fink install ipython-py25
  7. (Optionally) install CCPN 
    http://sourceforge.net/projects/ccpn
  8. (Optionally) install Wattos 
    http://sourceforge.net/projects/wattos
  9. (Optionally) install What If 
    Google it.
  10. (Optionally) install Xplor 
    Google it.
  11. (Optionally) install Procheck NMR 
    Google it.
  12. (Optionally) install Aqua 
    From http://sourceforge.net/projects/aquad
  13. (Optionally) install Povray 
    fink install povray
  14. (Optionally) install Molmol 
    Google it.
  15. (Optionally) install Molmol 
    Google it.
  16. Setup Eclipse
    Always start from command line instead of Dock/Taskbar so that it gets your environment settings such as PRODIR for the PROCHECK_NMR installation.
    CING is being developed on Eclipse 3.3 with PyDev Extensions for automatic code analysis and organization.
  17. Setup CVS archive
    Register with sourceforge to get a user account. Check out the CING project from CVS in Eclipse.
    This is a two wizard step process; first the CVS then the Pydev wizard.
